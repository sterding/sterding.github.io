--- layout: post title: how to empty / reset an existed file? date: '2012-04-12T11:21:00.003-04:00' author: Xianjun Dong tags: - linux - touch modified_time: '2012-05-09T10:17:56.098-04:00' blogger_id: tag:blogger.com,1999:blog-12445049.post-5094881309639461499 blogger_orig_url: http://onetipperday.blogspot.com/2012/04/how-to-empty-reset-existed-file.html --- Say, I have to create a new file regardless of the existence of the file.

"touch file" does not work in that case, because it will only update the time tag if a file is already existed.

One way I can think of is:

(rm file && touch file) || touch file

Other ways I got from this page (http://www.tech-recipes.com/rx/2993/how_to_empty_or_clear_the_contents_of_an_existing_unix_file/):

cat /dev/null > file.txt


or the simplest way:

>file.txt
