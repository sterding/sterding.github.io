--- layout: post title: bigWigAverageOverBed output order date: '2012-06-06T00:33:00.002-04:00' author: Xianjun Dong tags: - bigWigAverageOverBed modified_time: '2012-06-06T00:33:31.213-04:00' blogger_id: tag:blogger.com,1999:blog-12445049.post-733168037981109889 blogger_orig_url: http://onetipperday.blogspot.com/2012/06/bigwigaverageoverbed-output-order.html --- Be aware of the order of bigWigAverageOverBed output!

If the input bed file is short, it's fine - same order of input bed file.

If the input bed file is large (not sure how large will trigger this), the programme will sort the input by chromosome (e.g. chr1, chr2, ...) first and then output in order of sorted chromosome. Be careful of this!
