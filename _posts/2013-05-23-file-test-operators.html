--- layout: post title: File test operators date: '2013-05-23T13:10:00.001-04:00' author: Xianjun Dong tags: - bash modified_time: '2013-05-23T13:11:12.041-04:00' blogger_id: tag:blogger.com,1999:blog-12445049.post-4733471749692212725 blogger_orig_url: http://onetipperday.blogspot.com/2013/05/file-test-operators.html --- File test operators:

-e
file exists

-a
file exists (same as -e, but deprecated)

-f
file is a regular file (not a directory or device file)

-d
file is a directory

-h (or -L)
file is a symbolic link
