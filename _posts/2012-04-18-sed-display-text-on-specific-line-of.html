--- layout: post title: SED - display text on specific line of text file date: '2012-04-18T11:44:00.004-04:00' author: Xianjun Dong tags: - linux - more - sed - tail modified_time: '2014-05-22T11:24:49.406-04:00' blogger_id: tag:blogger.com,1999:blog-12445049.post-6066785062524009379 blogger_orig_url: http://onetipperday.blogspot.com/2012/04/sed-display-text-on-specific-line-of.html --- # print line number 52
sed -n '52p' # method 1
sed '52!d' # method 2
sed '52q;d' # method 3, efficient on large files 

sed -n '52,60 p' file # display line from 52 to 60

more +4 file # display line from 4 to the end

or

tail -n +2 file # display line from 2 to the end
