--- layout: post title: comment multiple lines in vim date: '2012-07-07T17:42:00.002-04:00' author: Xianjun Dong tags: - vi modified_time: '2012-07-07T17:42:13.500-04:00' blogger_id: tag:blogger.com,1999:blog-12445049.post-1254737763170726446 blogger_orig_url: http://onetipperday.blogspot.com/2012/07/comment-multiple-lines-in-vim.html --- Short answer: shift+v and up/down arrow to select lines to comment, then press :s/^/# / and Enter

NB: When you press : vim will add '<,'> to the command line so the final command is:
:'<,'>s/^/# /
